# Home-Loan-Optimiser
Home Loan Optimiser is a basic tool to help you visualize and quickly see the progress of you loan over its life time

## Features:
- Export and Import your loan details to save to a offline backup. (no data save on site)
- Generate Reports
- See pay off period over years & months. 

## 📂 Import / Export
Use Export to save all your current settings as a CSV file. Use Import to load them back — on any device, or to share with a partner or broker. The file contains two lines: field names on line 1, values on line 2. Use Report to download a formatted HTML summary you can print or email.
